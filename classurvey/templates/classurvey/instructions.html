Information for annotation process

Instructions:

  • Guidelines for annotation process:
    1. Select the most appropriate category for each of the ~100 sounds that you'll be presented one by one.
    2. During annotation choose a confidence level and write a comment, if necessary.
  • The annotation takes approximately 20-40 minutes. We suggest completing it in a single sitting.
  • It is preferable to be located in a quiet environment and use headphones so you can listen to the sounds without interferences.
  • To ensure accurate results, please refrain from using the browser's back button.
